# Sentence Builder

Develop an AI-powered program that constructs meaningful sentences from a given string of letters. The program should intelligently insert special characters (like commas, periods, etc.) where appropriate to enhance readability and coherence. The aim is to create a tool that can take any sequence of letters and generate a plausible and grammatically correct sentence.

## Requirements
* <strong>Sentence Generation:</strong> A sentence using words that start with each letter in the sequence (see examples below).
* <strong>Special Character Insertion:</strong> The AI should insert special characters to enhance readability.
* <strong>Grammatical Correctness:</strong> Ensure the generated sentence follows basic grammatical rules.

## Additional Challenge
* <strong>Custom Themes:</strong> Allow users to specify themes or constraints for the generated sentence (e.g., animals, food, actions).

## Examples
* `E, A, D, G, B, E` &raquo; Eddie Ate Delicious Grapes Before Eating
* `R, A, B, B, I, T` &raquo; Rats Are Biting Big Insects Together
